Funcionalidades WhereAreMyFriends - FindUs
 - Obter a LOCALIZAÇÃO ATUAL e possibilitar o usuário de SALVAR sua localização ou compartilhar com algum AMIGO e/ou GRUPO
 - SOLICITAR localização de amigo mediante a aprovação conforme COMFIGURACAO DE COMPARTILHAMENTO. 
 - Compartilhar com amigo localização por tempo pré-definido|indeterminado mediante a aceitação de ambas as partes.
 - Compartilhar com amigos|lista|grupo localização enviando alguma mensagem.
 - Controle de zoom do mapa com visualização dos usuários com ultimas informações disponibilizadas.
 - Usuarios possuem amigos relacionados pelo numero do telefone.
 - Lista de amigos com visualização da ultima atualização.
 - Gerência de grupos de amigos 
 - Perfil de Usuário
 - Compartilhamento de localizaçao|comentario de local para amigo|lista de amigos|grupo|REDES SOCIAIS.
 - Adicionar amigos atravez de contatos da agenda telefônica
 - Configurar dispositivo para obter localização automáticamente conforme frequência.
 - Localizar amigos distantes por no máximo alguma distância pré definida.
 - Alertar amigos quando raio de distÂncia entre amigos for menor que o suficiente
 
 IMPROCEDENTES
 - Cada usuário pode possuir diversos dispositivos (aparelhos) registrados para o mesmo usuário, 
   desta forma a localização deve ser identificada de acordo com o dispositivo sendo utilizado.
 
 
 
 LOGIN
  - O usuario deverá entrar com username ou email e com uma senha para entrar no sistema.
  - Caso o usuario nao possua cadastro, é possivel criar novo cadastro preenchendo as informações necessárias apartir do dispositivo no qual esta conectando.
  - Configuração de login automático para utilizar ultimo usuário da sessão para realizar o login.
